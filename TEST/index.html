Qual è la differenza tra i ruoli di HTML, CSS e JavaScript?
La differenza tra HTML, CSS e Javascript è dettata dai ruoli.HTML è un linguaggio di markup, con esso costruiamo l'interfaccia del nostro sito web, CSS è un linguaggio di stile, con quest'ultimo possiamo migliorare lo stile della pagina e Javascript è un linguaggio di programmazione, grazie ad esso riusciamo a dare una logica alla nostra pagina, aggiungendo eventi.
Quali sono i tag HTML di base?
I tag HTML di base, o detti semantici, sono molteplici. Prima di tutto vengono chiamati semantici perchè non contribuiscono alla resa visiva della nostra pagina, ma servono per scrivere il codice in maniera piu efficace e serve al compilatore del browser a capire il contenuto di questi tag.
Quali sono i principali elementi JavaScript che conosci ad ora?
I principali elementi Javascript visti fin ora sono: console.log() per mettere in output il risultato del nostro codice, visibile sulla console del browser; let e var per dichiarare variabili, c'è una differenza tra i due, dato che dichiarando una variabile con var, la variabile sarà visibile in tutto il codice, anche nei suo blocchi (if, while, do-while, for...), mentre con let questo non succede; prompt per inserire in input un dato. ParseInt per trasformare un dato in un intero.
Qual è la differenza tra una tipizzazione rigida e una debole?
I linguaggi con tipizzazione forte richiedono una dichiarazione obbligatoria dei tipi di dati, altrimenti il codice genera un errore. Al contrario i linguaggi a tipizzazione debole non richiedono obbligatoriamente una tipizzazione dei dati.
Qual è il punto di collegamento tra back end e front end e come vi comunichiamo.
Il collegamento tra front-end e backend avviene tramite il tag form, aggiungendo l'attributo action e un metodo, che sia post o get.
